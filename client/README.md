A Project for Creating IT Department logs.

Features:

- add Logs
- edit logs
- if a log needs Attention/Pending ( red: true, blue: false)
- delete logs
- search logs
- add Technician
- delete Technician
- view Technician


'json-server' as Dev Dependency and Dummy Backend Server
'Materialize-css' for main styling
